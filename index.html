01 / ABOUT

このページについて

「マルチSNS参謀」は、中小企業の経営者・販促担当者・個人クリエイターのための、SNS戦略設計＋月次伴走サービスです。マーケティングの専門知識がなくても、神崎潤（かんざき じゅん）という参謀役のガイドに従うだけで、何を・どこで・いつ・どう発信するかが組み上がります。

このサービスで手に入るもの

  • 自社にぴったりのSNS組み合わせ（やる／やらないの判断つき）
  • 2年戦略カレンダー・6ヶ月ロードマップ・1ヶ月運用表の3階層プラン
  • 北極星指標（最終的に追う1つの数字）とその測り方
  • 来週から動ける「今週やる3タスク」リスト

逆に、こんな方には向きません

  • すぐに広告で売上を立てたい（本サービスはオーガニック運用が主軸）
  • 1ヶ月でフォロワー1万人を狙いたい（誇大な約束はしない方針）
  • すべて丸投げしたい（伴走型のため、月1回の打ち合わせには参加が必要）

03 / METHOD

マルチSNS参謀メソッドとは

ダイレクトレスポンスマーケティング（成果を測れる形で発信を設計する手法）の考え方を、SNS運用に落とし込んだ v2.0 では6フェーズ構成。素人でも答えられる7問のヒアリングから始まり、最後は来週月曜の朝に動ける3タスクまで届けます。v2.0 Phase 1.5 追加

PHASE 0

ヒアリング7問

ビジネス内容・お客様像・ゴール・使えるリソース・候補SNS・NG事項を30分で整理。マーケ用語ゼロ＋例示付き設計です。

PHASE 1

サービス設計層

コンセプト・ターゲットペルソナ（仮想の代表顧客）・USP（他社にない強み）・3C分析・北極星指標を1日で固めます。

PHASE 1.5

オファー＆ファネル設計

商品ピラミッド（無料→お試し→本商品→高単価）、リードマグネット3案、Irresistible Offer 5要素、CTA文言テンプレ10種、計測設計（LTV×許容CAC）まで。Kennedy/Abraham系譜のDRM中核を組み込みます。

PHASE 2

マルチSNS戦略マップ

8つのSNSから「メイン1・サブ2・補助任意・見送り」を判定。プラットフォーム間の導線（誰をどこへ誘導するか）も設計します。

PHASE 3

3階層ロードマップ

2年カレンダー・6ヶ月詳細・1ヶ月詳細の入れ子構造で、長期と短期の両方が同時に見える地図を作ります。

PHASE 4

月次運用カレンダー（A/Bテスト前提）

選定SNSごとに「いつ・何を・どんな形式で投稿するか」を月16本〜の粒度で表化。各投稿にフックA/B案・サムネ2方向を組み込み、初日からテスト文化を埋め込みます。

PHASE 5

実行＋振り返り設計（撤退KPI数値化）

今週やる3タスク・チェックポイント・撤退判断基準（CAC/LTV比30%超・登録率1%未満等の数値閾値5項目）まで明文化。動き続ける仕組みを残します。

v2.0 の進化ポイント（v1.0からの差別化）

  • Phase 1.5 新設：SNSは「投稿を続ける場」から「お客様を獲得する装置」へ。商品ピラミッド／リードマグネット／Offer 5要素／CTA10種／計測設計まで一気通貫
  • 撤退判断を数値で：「気合で続ける／なんとなく辞める」を排除。CAC/LTV比・転換率・保存率の5閾値で客観判定
  • A/Bテスト前提：初日からフック2案・サムネ2方向。Kennedy流テスト文化を運用カレンダーに埋め込み
  • 分割出力＋進行ゲージ：長尺出力の途中切断を防止。再現性と納品品質を担保
  • 表セル文体ルール除外・自己チェック工程：納品物の自然な日本語と品質ゲートを両立
